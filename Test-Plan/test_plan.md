Sample Web Application

- UI testing
- Basic usability testing

## Test Types
- Smoke Testing
- Functional Testing
- Regression Testing

- Browser: Chrome, Firefox
- OS: Windows 10

## Entry Criteria
- Application is deployed to test environment
- Test data is available

## Exit Criteria
- All critical and major bugs are fixed
- Test cases are executed successfully
